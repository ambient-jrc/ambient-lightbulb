Yeelight LED Demo App

    Team JRC Yeelight App

  • This app has multiple functions that control a Yeelight bulb.

    Function 1: Looks up the current temperature of Honolulu and sets the color temperature of the bulb to be blue or orange. Blue being colder while orange being warmer.
    Function 2: A sleep function that turns the bulb off after a set amount of time. (Either 5 or 30 seconds)
